GitHub 반영 안내

1. 파일 구성

./deadline-tasks.json 파일을 이 HTML과 같은 폴더에 두세요.

/docs ├─ deadline-calendar.html └─ deadline-tasks.json

파일명을 바꾸면 HTML의 DATA_URL 값도 같이 바꾸면 됩니다.

2. JSON 작성 규칙

큰따옴표 사용 마지막 쉼표 금지 날짜는 YYYY-MM-DD priority: high/medium/low state: todo/doing/hold/done

GitHub Pages에서는 JSON 업로드 후 새로고침하면 바로 반영됩니다. 반영이 늦으면 상단의 “데이터 다시 불러오기”를 눌러 주세요.
